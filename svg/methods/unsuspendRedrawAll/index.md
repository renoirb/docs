## <span>Notes</span>

### <span>Remarks</span>

Typically, you should call the **unsuspendRedrawAll** method as the last call of a set of SVG DOM calls to make sure that all pending [**suspendRedraw**](/svg/methods/suspendRedraw) method calls are canceled.
