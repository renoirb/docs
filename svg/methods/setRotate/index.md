## <span>Notes</span>

### <span>Remarks</span>

The **setRotate** method can raise a [**DOMException**](/dom/DOMException) exception with the code NO\_MODIFICATION\_ALLOWED\_ERR. This exception is raised if you try to change the value of a read-only attribute.
