## Summary

Lists the methods supported by the target resource.

## Overview table

Direction
:   Response

Required
:   Along with 405 Method Not allowed

Features
:

## Syntax

    Allow = #method

## Examples

```
Allow: GET, HEAD, PUT
```

```
Allow: GET, HEAD, PUT, DELETE, POST
```

## Usage

     Must be sent in a 405 Method Not Allowed response.

*Should* be sent to an OPTIONS response.
