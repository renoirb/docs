### <span>Summary</span>

Returns value of a string converted to uppercase while keeping the original string intact.

### <span>Syntax</span>

``` js
"string".toUpperCase
```

### <span>Parameters</span>

None

### <span>Example</span>

``` js
var userInput="MixEDString";
document.write(userInput.toUpperCase());
```
