## <span>Summary</span>

Creates a structured clone of the value parameter.

Method of [apis/indexeddb/IDBCursor](/apis/indexeddb/IDBCursor)[apis/indexeddb/IDBCursor](/apis/indexeddb/IDBCursor)

## <span>Syntax</span>

``` js
var object = object.update(value);
```

 Data-type
