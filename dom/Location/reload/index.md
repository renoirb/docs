## Summary

Reloads the current document.

Method of [dom/Location](/dom/Location)[dom/Location](/dom/Location)

## Syntax

``` js
 location.reload(ignoreCache);
```

## Parameters

### ignoreCache

 Data-type
:   Boolean

 Whether to ignore the cache and reload the page from scratch.

## Return Value

No return value
