## Summary

Multiplies the value of two expressions.

## Syntax

    result = number1 * number2

**result**
:   Any variable.

**number1**
:   Any expression.

**number2**
:   Any expression.

## Examples

``` js
var x = 5;
var y;
y = x * 2; // result: y = 10
```
