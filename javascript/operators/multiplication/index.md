## <span>Summary</span>

Multiplies the value of two expressions.

## <span>Syntax</span>

<span class="language">JavaScript</span>

    result = number1 * number2

**result**
:   Any variable.

**number1**
:   Any expression.

**number2**
:   Any expression.

## <span>Examples</span>

``` js
var x = 5;
var y;
y = x * 2; // result: y = 10
```
