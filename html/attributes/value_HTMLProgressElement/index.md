## Notes

### Remarks

When the **value** attribute is omitted, the [**progress**](/html/elements/progress) element becomes indeterminate, that is, it shows activity but not how much progress has actually been made. If the **value** attribute is used without a [**max**](/html/attributes/max(HTMLProgressElement)) value, the range is from 0 to 1.
