## Examples

This example uses the **FRAME** attribute to render different border frames.

``` html
<TABLE FRAME="above">
<TR><TD>A table with</TD><TD>the value of frame set to "above"</TD></TR>
<TR><TD>cell</TD><TD>cell</TD></TR>
</TABLE>
```
